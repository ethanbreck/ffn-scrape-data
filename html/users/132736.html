Author has written 57 stories for UFO.

I am a lifelong UFO fan who enjoys most sci-fi as long as there's a bit of romance thrown in. I can't help it! I'm a hopeless romantic hidden inside a cynical person. What do you do?

I have written UFO stories since first watching the show as a tween back in . . . well, a long time ago! I never saved those early efforts, but many of the original basic plots have found their way into the stories I've written and kept. Here's the breakdown for my stories, since Fanfiction doesn't allow me to keep them all together with their series or sagas.

Conover Saga: These stories can be found in numbered order with no prefix to each number. 1-20. Please note: #21 is a Next Gen story about the children from the saga.

Devilsgate Saga: These stories can be found in numbered order with DS as the prefix to each one. 1-6.

Shakespeare Saga: These stories can be found in numbered order with SS as the prefix to each one. 1-5. (so far!)

The Artist Series: These stories can be found in numbered order with AS as the prefix to each one. 1-3.

The Interpol Series: These stories can be found in numbered order with IS as the prefix to each one. (only the first is finished so far!)

The Writer Series: These stories can be found in numbered order with WS as the prefix to each one. (only the first is finished so far!)

The Uphoria Series: These stories can be found in numbered order with US as the prefix to each one. 1-2. (so far!)

The Gaia Series: These stories can be found in numbered order with GS as the prefix to each one. 1-4. (so far!)

The rest of my stories are stand-alones and have no bearing on other stories I've written, so they can be read in any order. I hope you'll enjoy reading my stories as much as I enjoyed writing them!

My favorite quote? It's one of my own: The sky is not the limit, nor are the stars.
