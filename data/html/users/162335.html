Only Joking: Sort of a prequel to Always a Bridesmaid, Never a Bride, but not really. It's set in the same Universe but it can be read seperately. People kept begging and begging for me to write a sequel and I coudn't think of a plot line that hasn't been done before, so I'm doing a prequel with Alicia and George as the main characters. I came up with this a while ago, and never really had the courage to write it. After reading some of the stories about Ginny leaving, I thought it was well time! ~IN PROGRESS~
