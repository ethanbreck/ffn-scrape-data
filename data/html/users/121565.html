Hi everyone, I'm Nissa. Well, that's not my real name, but half the fun in posting here is getting to make up some kind of pen name. So Nissa it is! Ha!

I started this account ages ago. I don't really write much fanfic anymore because I tend to focus mainly on art now. It got to the point where I was making up stories solely so I could draw scenes from them! So yes, these stories are definitely older (and because they are older, I don't feel they're quite up to snuff with what I could currently write). The Star Wars ones were written before Attack of the Clones came out, so if there are any conflicts with the movies then that's why. Who knows, in the future I might write more fanfic, but for now this is it. I had fun writing these at the time, so I won't be deleting them. Just, you know, be gentle if you need to criticize. ;)

Looking for my art? Try either my main website, Run to the Ocean.net, or my deviantArt gallery.

Oh, and if you've been around in the Star Wars or Lord of the Rings fandom for a while, I'm the same person who used to run Jedi Girl - SSR and Ithildin, so it's entirely possible that we may have bumped into each other before. ;)
